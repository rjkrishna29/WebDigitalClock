## ✨ Features

-   ✅ **Real-Time Digital Clock:** Displays the current local time, updating dynamically every second.
-   ⏱️ **Precision Stopwatch:** Includes **Start**, **Stop**, and **Reset** functionality for accurate time measurement.
-   ⏳ **Versatile Countdown Timer:** Set custom countdowns with **Start**, **Pause**, and **Reset** controls.
-   📱 **Fully Responsive Design:** A fluid layout that ensures a seamless experience on desktops, tablets, and mobile devices.
-   🎨 **Clean & Minimalist UI:** A modern, intuitive interface designed for clarity and ease of use.

---

## 🛠️ Technologies Used

This project was built from the ground up using core web technologies:

-   **HTML5** for the structure and content.
-   **CSS3** for styling, layout, and responsiveness.
-   **Vanilla JavaScript** for all the logic and functionality.
